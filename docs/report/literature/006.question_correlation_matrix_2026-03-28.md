# Paper-to-Question Correlation Matrix

**Date**: 2026-03-28
**Purpose**: Map the currently indexed literature corpus onto the main research questions in `docs/research_plan.md` so the review can answer both “what does each paper say?” and “why is it in this repo?”.

## Research question keys

- **RQ1** — immediate bottleneck removal on the DSA path
- **RQ2** — framework redesign for the nanosecond regime
- **RQ3** — end-to-end RPC decomposition and adaptive offload
- **RQ4** — generalization across hardware and software domains
- **RQ5** — multi-accelerator composition and deployment paths
