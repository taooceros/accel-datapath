# Expanded paper folder index

**Date**: 2026-04-13
**Type**: Literature corpus expansion index
**Purpose**: extend the paper-folder workflow from the original seeded six-paper set to the broader literature-report corpus that already had local PDFs in `papers/top_tier_pdfs/`.

## What changed in this pass

This pass added per-paper folders for the literature-report papers that were already locally downloaded but did not yet have stable homes under `docs/report/literature/papers/`.

Each added folder now contains:

- `paper.pdf`
- `paper.md`
- `README.md`

The reusable extractor from `tools/paper-text/` was used to generate searchable Markdown. For this broader batch, the safe default was page-based `paper.md`; sectionization was not forced across the whole set because the current verifier remains tuned to papers with more recoverable section headings.
