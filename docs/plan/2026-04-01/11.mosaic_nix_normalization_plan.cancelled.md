# Plan: Normalize Mosaic Visualization Through Nix

**Date**: 2026-04-01

**Status**: cancelled on 2026-04-03; superseded by the Observable Framework rewrite plan.

## Goal

Update the newly added Mosaic report visualization so environment management and invocation flow through nix/devenv rather than unmanaged local installs.

## Scope

### In scope

- inspect the current `devenv.nix` setup
- add the required Node/tooling support for the Mosaic app through nix/devenv
- update the Mosaic README and report references to use nix-managed commands
- remove generated local dependency directories that should not be relied on

### Out of scope

- replacing existing Python plotting flows
- introducing a larger frontend monorepo structure

## Deliverables

- nix/devenv-managed support for building/running `tools/mosaic-tonic-report`
- documentation that uses `devenv`-based commands
- verification that the dashboard still builds into `docs/report/artifacts/003.tonic_bounded_matrix_mosaic/`
